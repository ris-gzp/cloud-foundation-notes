# Linux Cheat Sheet

## Filesystem Layout
- / : root
- /home : user files
- /etc : config files
- /var : logs
- /tmp : temporary files
- /usr : user programs

## Must-Know Commands
- pwd, ls, cd, cp, mv, rm
- cat, less, grep, find
- wc, sort, df, du

## Permissions
- chmod 600 = owner read/write only
- chmod 644 = owner rw, others read
- Never run as root!
